# AbeloHost Shop

### Getting Started

First, run the development server:

```bash
npm install
npm run dev
```
### Technical stack

- TypeScript
- NextJS (App Router)
- Zustand
- Axios
- SCSS-модули
- Prettier
- ESLint
- Stylelint

### Folder structure

- FSD

### API for requesting data

- API DummyJSON (https://dummyjson.com/)

### Building and launching the application

- Docker

```bash
# build
docker build -t abelohost-shop .

# start
docker run -p 3001:3000 --rm abelohost-shop
```
